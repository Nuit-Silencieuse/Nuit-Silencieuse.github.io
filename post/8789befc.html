Linux操作系统的基本使用

Created2024-07-31|Updated2024-09-10
|Post Views:

这篇文章是WSL的Ubuntu发行版的基本使用。

常用命令整理

网络

ip address / hostname -I：查看本机网络ip地址。注意是以太网接口（通常是eth0）的IPv4地址。

监控

ps -ef | grep <name>：查看并筛选与此name有关的进程，该进程名可以是进程的全部或者部分。

top （普通版本）/ htop（彩色增强版本）：显示系统中的进程和它们的CPU、内存使用情况。按q退出。

sudo service SERVICE_NAME status：查看某服务的运行状态，以及服务的输出信息。

终端

vim ~/.bashrc：bashrc是”bash run command”的缩写，即运行命令的脚本文件，用于配置bash shell的环境和设置，保存一些个性化设置，如命令别名、路径等。为每一个运行bash shell的用户执行此脚本文件。当bash shell被打开时，该文件被读取。

source ~/.bashrc：每次修改.bashrc后，使用source ~/.bashrc（或者 . ~/.bashrc）就可以立刻加载修改后的设置，使之生效。
